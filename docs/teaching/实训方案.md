# ODPlatform 实训方案

## 目标
通过实战掌握目标检测项目的完整开发流程。

## 课程安排

### D1: 架构设计
- Monorepo 结构介绍
- 分层架构理解
- 命名规范学习

### D2: 环境搭建
- Conda 环境创建
- 依赖安装
- 项目初始化

### D3-D4: 配置管理
- Pydantic 配置模型
- YAML 配置加载
- 配置合并与生成

### D5: 数据转换
- VOC 格式解析
- YOLO 格式生成
- COCO 格式支持

### D6: 数据验证
- 数据集结构检查
- 标签格式验证
- 统计分析与可视化

### D7: 训练编排
- 训练参数配置
- 训练流程执行
- 检查点管理

### D8: 模型评估
- 验证集评估
- 指标计算
- 报告生成

### D9: 推理流水线
- 批量推理
- 结果后处理
- 结果导出

### D10: 项目总结
- 代码审查
- 文档完善
- 成果展示
